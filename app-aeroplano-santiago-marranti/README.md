# Base Angular + Express

Proyecto base para aprender conexion entre frontend y backend.

## 1) Backend

```bash
cd backend
npm install
npm run dev
```

El backend queda escuchando en `http://localhost:3000`.

## 2) Frontend (Angular)

En otra terminal:

```bash
cd frontend
npm install
npm start
```

El frontend queda en `http://localhost:4200`.

## 3) Probar flujo

1. Abri `http://localhost:4200`.
2. Hace click en **Pedir mensaje al backend**.
3. Vas a ver un cartel (`alert`) con el mensaje del backend y el mismo texto en pantalla.
